<div align="center">

<img src="https://readme-typing-svg.demolab.com?font=Fira+Code&weight=300&size=12&duration=4000&pause=2000&color=9d6fff&center=true&vCenter=true&width=720&height=22&lines=Branch+%E2%86%92+Complete+Template+%E2%86%92+Test+%E2%86%92+Commit+%E2%86%92+Push+%E2%86%92+Pull+Request" alt="Workflow" />

</div>

```bash
# 1. Create your branch
git checkout -b your-name/pf-what-you-are-adding

# 2. Pick a template from examples/ and complete it

# 3. Test before committing
g++ -Wall -std=c++17 your-file.cpp -o test && ./test   # C++
python your-file.py                                     # Python

# 4. Save your completed file directly in programming-fundamentals/
#    Do NOT push back into examples/

# 5. Commit and push
git add .
git commit -m "feat: add completed [topic] in [cpp/python]"
git push origin your-name/pf-what-you-are-adding
```

**File Naming:**
```
kebab-case only

✅  hello-world.cpp       bubble-sort.py
❌  HelloWorld.cpp        BubbleSort.py
```

**Required header on every file:**
```cpp
/*
 * File:    your-file.cpp / your-file.py
 * Author:  Your Name (@github-username)
 * Date:    2026
 */
```
